# Introduction
*exch-rate* is a tool to fetch exchange rates and convert foreign currencies.

# Installation
To install hotspotd, just follow these steps:
```
pip3 install exch-rate
```

To uninstall hotspotd, just say:

```pip3 uninstall exch-rate```

# Usage

```exch-rate usd inr```

outputs:

67.860001
